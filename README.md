Abaixo especificações e ferramentas usandas no desenvolvimento do Aplicativo:

Aplicativo mobile feito com React Native + Firebase para ajudar no controle financeiro pessoal.

## Sobre

O Meu Limite foi criado para facilitar o acompanhamento de gastos do dia a dia.

Com ele é possível:

- Criar conta com email e senha
- Registrar gastos
- Editar despesas
- Visualizar despesas no dashboard
- Organizar melhor o limite financeiro mensal

A proposta é ajudar no controle do dinheiro de forma simples e prática.

## Tecnologias

- React Native
- Expo
- Firebase Authentication
- Firestore
- React Navigation

## Funcionalidades

✅ Cadastro e login com email e senha

✅ Tela de boas-vindas para novos usuários

✅ Dashboard com acompanhamento de gastos

✅ Cadastro de despesas

✅ Edição de despesas
